POST #1 - MAKING THE SITE

so, yeah, i guess i have a blog now...i think it'll be a good use of my time if i remember it.
i don't really have anything to talk about, so i'll run you through how i made the site.
this site is literally held together by spaghetti, so i'll keep this post to just text.

i decided to make this site when i was scrolling reddit (i should be using lemmy instead) when i saw
a post shilling neocities. and i was like, fuck it, why not?
honestly, i've never been a fan of the web 1.0 designs this site is a fan of. i'm too young to remember
web 1.0, so i get the vibe, but it just doesnt look good.
dont get me wrong, i get why people like it and i have found some sites i like, but...
i prefer the minimalist, space-relative style i've coded here.
just looks better to me.

anyways, first step on making the site was making the homepage. and as somebody who uses vsc...
the built in neocities code editor is bad.
i beg of you, *use vsc*. its way better. it's going to be annoying setting this site up so it isnt just
local, but whatever.
